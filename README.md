# QurbaniHat – Livestock Booking Platform

QurbaniHat is a modern livestock marketplace where users can explore and book animals (cows and goats) for Qurbani. The platform provides a seamless experience for browsing healthy, verified animals and placing bookings after secure authentication.

## ✨ Key Features
- **Modern UI/UX:** Clean, responsive design with premium aesthetics.
- **Animal Catalog:** Browse a wide variety of cows and goats with detailed information.
- **Authentication:** Secure Email/Password and Google Social Login using `Better Auth`.
- **Sorting:** Sort animals by price (Low to High / High to Low).
- **Animal Details:** Comprehensive view of animal age, weight, breed, and location.
- **Booking System:** Easy-to-use booking form for registered users.
- **User Profile:** Manage your profile and update information seamlessly.
- **Animations:** Smooth entrance and micro-animations using `Animate.css`.
- **Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop.

## 🛠️ Technology Stack
- **Frontend:** React.js, Vite
- **Backend:** Node.js, Express.js
- **Authentication:** Better Auth
- **Database:** MongoDB
- **Styling:** Vanilla CSS
- **Icons:** Lucide React, React Icons
- **Notifications:** React Hot Toast
- **Animations:** Animate.css

## ⚙️ Setup Instructions
1. Clone the repository.
2. Install dependencies for both frontend and backend:
   ```bash
   npm install
   cd server && npm install
   ```
3. Create a `.env` file in the `server` directory and add your credentials (MONGODB_URI, GOOGLE_CLIENT_ID, etc.).
4. Start the backend server:
   ```bash
   cd server && node index.js
   ```
5. Start the frontend:
   ```bash
   npm run dev
   ```
